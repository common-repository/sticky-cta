Flexible sticky cta button creator allowing you to stick floating buttons to the side of your site.

== Description ==

This simple plugin having high converting sticky call to action floating buttons sticks at the side of your page as the user scrolls..  

= Use it for sticking information like: = 
* Phone number
* Email address
* Social icons
* Requied CTA button
